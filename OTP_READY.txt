================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: OTP not arriving
  Solution:
  1. Check server is running: cd server && npm start
  2. Check phone number format: +91XXXXXXXXXX
  3. Check phone is in Twilio trial list
  4. Wait 60+ seconds before retrying

Problem: "Too many requests" error
  Solution:
  1. Wait 60 seconds before requesting another OTP
  2. Use different phone number
  3. Check server logs for exact wait time

Problem: "Server not responding"
  Solution:
  1. Ensure server is running: cd server && npm start
  2. Check port 4000 is not in use
  3. Restart server

Problem: "Twilio not configured"
  Solution:
  1. Check server/.env exists
  2. Verify it has all 3 credentials
  3. Restart server
