TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Q: "OTP sent" but no SMS received?
A: Server not running. Check Terminal 1 shows "Server listening on port 4000"

Q: Server shows "Twilio configuration missing"?
A: Create server/.env with Twilio credentials

Q: "Cannot find module 'twilio'"?
A: Run: cd server && npm install

Q: "Port 4000 already in use"?
A: Kill process: taskkill /PID <PID> /F

Q: Server running but still no SMS?
A: 1. Check phone number format: +91XXXXXXXXXX
   2. Verify Twilio credentials in server/.env
   3. Check Twilio account has credits
   4. Check server console for error messages
